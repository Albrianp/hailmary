Welcome to My Universe. I Build Things with Code.

Saya Albrian, seorang kreator digital yang hobi mengeksplorasi batas antara seni visual dan fungsionalitas sistem. Website ini adalah ruang bermain sekaligus pembuktian teknologi tempat saya menyatukan baris logika menjadi sebuah mahakarya interaktif.

01 / FRONTEND CRAFTSMANSHIP

HTML & JavaScript Exploration.

Tempat saya meracik antarmuka modern. Menggunakan struktur HTML yang bersih dan ekosistem JavaScript tingkat lanjut (seperti Three.js di web ini) untuk menghidupkan elemen visual, menciptakan animasi kosmik, dan membuat website yang merespons setiap gerakan kursor Anda secara instan.

02 / INTELLIGENT LOGIC

The Power of Python Scripting.

Di balik keindahan visual, ada logika yang bergerak. Menggunakan Python, saya membangun arsitektur belakang layar yang cerdas, mulai dari pemrosesan data, otomatisasi skrip, hingga manipulasi algoritma untuk memastikan setiap sistem berjalan dengan otak komputasi yang kuat.

03 / SYSTEM AUTOMATION

Bash Scripting & Server Command.

Bekerja langsung di jantung sistem operasi. Dengan skrip Bash, saya mengendalikan terminal untuk mengotomatiskan tugas-tugas rumit, mengoptimalkan kinerja server, dan mempercepat manajemen infrastruktur agar segala sesuatu berjalan secepat cahaya.
